# Guerra Consórcios's Landing Page
Landing Page made for Guerra Consórcios. A basic simulation page is implemented too. Features a mobile version for both.

## Stack
React - Front-end
Some Figma

## Main Libs
Tailwind CSS
React Router Dom
Some visual libs

## Getting started

```sh
yarn install
yarn start
```
